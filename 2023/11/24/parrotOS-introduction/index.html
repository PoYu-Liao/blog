取代kali輕量化滲透測試作業系統parrotOS

發表於 2023-11-24 更新於 2024-07-28 分類於 Linux ， Penetration Test ， Security

硬體需求

CPU處理器: Dual Core CPU
Graphics: 不需要圖形加速功能
記憶體: 1 GB RAM
儲存空間: 至少20 GB可用空間

介紹

Parrot OS 是一個基於 Debian 的免費開源Linux發行版，主要設計用於測試、漏洞評估、渗透測試、數位鑑識和匿名上網等安全性相關的任務。Parrot提供了一個強大而靈活的環境，專為安全專業人員、測試人員和注重隱私的人員而設計。Parrot內建Kali Linux的常見的工具之外，還有自己衍生的工具如AnonSurf、Airgeddon、Wifiphisher...等。該系統的設計初衷是讓安全專家熟悉並讓新入門的學生易於使用，但它並沒有像其他通用發行版那樣試圖隱藏其內部結構。這意味著系統中包含了許多自動化工具，使其更易於使用，同時也很好地暴露了系統的底層內容。一個很好的例子是鸚鵡更新提醒：這是一個簡單但功能強大的程序，每週提示使用者檢查一次系統升級。但它不是將升級過程隱藏在進度條後面，而是從 apt 輸出向用戶顯示完整的升級過程。另一個重要的區別是，Parrot 預設會停用系統中預先安裝的所有網路服務，不僅是為了保持非常低的 RAM 佔用空間並提供更好的效能，而且還可以避免目標網路中的服務暴露。每個網路服務都需要在使用者需要時手動啟動。此外parrot有分為多個版本：Security、Home、Cloud...等版本，以滿足所有可能的硬體和使用者的需求。

安裝教學(以下示範以 Security Edition和Home Edition為主)

1.到官網下載所需版本的iso檔https://www.parrotsec.org/download/，其中左側欄位可以選擇版本，右側download為直接使用瀏覽器下載，torrent則是使用BT。

2. 選擇第一項try/install

  1. 選擇install Parrot(如果沒有要安裝只是想要嘗試的話，到這步即可)

  2. 接下來，選擇系統語言、區域跟輸入法

  3. 分割磁區和選擇是否使用swap

如果需要可以透過新增密碼來加密系統

  1. 設定完使用者帳戶後，確認配置都沒有問題就能按下install

  2. 完成
