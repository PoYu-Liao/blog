linux 版本更新指令

發表於 2024-07-28 分類於 Linux

本篇方法適用於linux所有版本的OS更新

更新OS或kernel層指令

sudo apt update
接下來指令請三選一使用

  1. apt upgrade -y

該apt upgrade命令下載並更新軟體包，而不會刪除 Kali Linux 系統上先前安裝的任何內容。但是，請注意，如果套件的新版本需要額外的相依(dependencies)項或刪除現有相依(dependencies)項，apt upgrade則會阻止此類套件。

  1. sudo apt full-upgrade

該apt full-upgrade命令下載並更新軟體包。如果需要，它也會刪除已安裝的軟體包。此命令對於可能需要新相依(dependencies)項的更重要的更新很有用，或者應刪除過時的軟體包以確保系統完全升級。主要推薦以這指令更新

  1. apt dist-upgrade -y

該apt dist-upgrade命令的作用與常規升級相同，同時智慧地處理不斷變化的相依(dependencies)項、刪除過時的軟體包以及添加新的軟體包。

其他更新方式

有些作業系統有自己獨有的更新指令，例如parrotOSsudo parrot-upgrade
