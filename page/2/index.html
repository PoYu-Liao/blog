橢圓曲線密碼學 Elliptic Curve Cryptography (ECC)

特色優點

  • 最早開始發展於1980年中期
  • 與 RSA 和離散對數相同的安全級別
  • 更少的計算量
  • 更短的簽名(signatures)和密鑰(keys)
  • trap-door function
  • 在同安全級別下，與RSA相比加密可以更快地使用短公鑰
